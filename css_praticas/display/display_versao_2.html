Displays


1. Inline

Elemementos que recebem esse valor, ocupam somente o espaço que for necessario. Não se pode definir width e height a eles. O documento é renderizado como parte do fluxo de texto da página e força quebra de linha.


2. Block

Elementos com esse valor, ocupam todo o width disponivel, caso não seja definido um automaticamente. Pode-se dar valores de width e height aos mesmos. Sua principal caracteristica, esta no fato de que sempre vai forçar uma quebra de linha em elementos antes ou depois dele

3. Contents

O valor contents, basicamente vai fazer com que a caixa de um elemento html, suma da ávore de renderização de uma página html
